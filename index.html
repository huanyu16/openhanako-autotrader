传统交易工具的四个死穴
你有没有遇到过这些情况？
😓

学不会

TradingView、Thinkorswim、Wind……每个工具都要学几个月。写指标要学 Pine Script，做回测要学 Python，门槛太高。

😱

管不住手

追涨杀跌、报复性交易、扛单不认亏。情绪化操作是散户亏钱的第一大原因，但没有工具帮你守住纪律。

⏰

盯不过来

美股半夜开盘，你在中国睡觉。错过开盘窗口、来不及止损、错过最佳买卖点，这些每天都在发生。

💸

策略不靠谱

各种「老师」推荐的策略，没有经过回测验证，没有风控，不知道什么时候该买什么时候该卖，全凭感觉。

核心策略
三个在历史上赚到过天量利润的交易员
他们的方法，现在内置在你的系统里
1907 — 1929
杰西·利弗莫尔
Jesse Livermore · 华尔街巨熊
  • 1907年一天赚300万美元（相当于今天数亿美元）
  • 1929年做空股市，赚1亿美元（当时美国GDP的1%）
  • 发明了突破交易法，被后世无数基金沿用
"钱是坐着等来的，不是操作来的。"
→ 突破交易法 · 阻力位突破 + 量能确认
1957 — 1959
尼古拉斯·达瓦斯
Nicolas Darvas · 舞蹈家交易员
  • $36,000 起步，18个月赚到 $2,250,000
  • 边全球巡演边交易，完全靠技术分析
  • 发明「箱体理论」，至今仍是最有效的趋势跟踪方法之一
"我只在箱体被突破时买入，这比任何基本面分析都可靠。"
→ 箱体理论 · 价格箱体突破识别
1987
拉瑞·威廉姆斯
Larry Williams · 短线交易之王
  • 1987年罗宾斯世界杯期货交易锦标赛冠军
  • $10,000 → $1,147,607（一年11,376%收益率）
  • 发明威廉指标(%R)，每个交易终端都内置了这个指标
"众人恐惧时贪婪，众人贪婪时恐惧。"
→ Williams %R · 超买超卖情绪判断
工作方式
一句话交易
系统自动完成剩下的一切
不需要学任何工具，不需要写任何代码。你只需要像和朋友聊天一样说你想做什么。
1

你说一句话

"买入100股苹果"
"看看NVDA的信号"
"熔断！全部撤单"

2

AI 分析策略

3

风控自动检查

仓位、止损、频率、日亏损……7项风控规则逐条验证，不通过坚决不下单

4

执行并报告

通过 Alpaca API 下单，实时返回成交确认，全程可审计追溯

核心能力
不只是交易，是一套完整的交易操作系统
🗣️

自然语言驱动

用中文/英文和系统对话，不需要学习任何交易软件。说"买入"、"卖出"、"分析一下"，系统自动理解并执行。

🧠

多策略融合

不是单一指标盲信，而是 Darvas + Livermore + Williams + RSI + 趋势分析多维融合，输出 -10 到 +10 的综合评分。

🛡️

七层风控拦截

白名单 → 做空限制 → 单笔限额 → 标的仓位 → 总仓位 → 日内亏损 → 交易频率。任何一关不通过，交易直接拒绝。

🤖

7×24 自动调度

开盘前扫描 → 开盘执行 → 午间检查 → 收盘报告 → 每5分钟止损监控。你睡觉的时候，系统在替你盯盘。

📊

完整审计追溯

每一笔交易、每一次风控拦截、每一次熔断，全部记录在 SQLite 审计日志中，随时可查可导。

🌍

多资产支持

美股、期货、加密货币、期权，一个系统覆盖所有资产类型。从模拟盘到实盘平滑过渡。

🚨 紧急熔断

一句"熔断"，所有挂单撤销，交易暂停。不需要在多个平台间切换操作。

📋 持仓风险概览

实时显示每个持仓的仓位占比、盈亏状态、是否超限，一眼看清整体风险。

🔒 标的白名单

设定可交易的标的范围，防止 AI 误操作买入不在计划内的股票。
